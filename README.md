# Проект `Movies`
Проект является дипломной работой на курсе Web-разработчик в Я.Практикуме.

## Функционал:
- лендинг с описанием дипломной работы,
- регистрация и авторизация,
- изменение данных пользователя,
- загрузка фильмов со стороннего api,
- поиск по фильмам.

- JavaScript,
- React,
- JSX,
- CSS,

- Flex,
